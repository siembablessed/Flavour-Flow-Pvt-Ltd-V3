# Flavor Flow - Wholesale Spirits

Premium spirits wholesale platform for retail and hospitality buyers.

## Features

- Product catalog with wholesale pricing
- VAT-inclusive unit pricing
- Cart functionality
- Transport quote requests
- Contact section

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn UI

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
